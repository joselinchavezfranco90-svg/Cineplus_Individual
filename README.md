# Cineplus_Individual
Aplicación Básica para un cine creada en visual studio code en la cual se utilizo lenguaje python, esta consta de 4 ventanas en las cuales puedes realizar compra de boletos y compra de productos como dulces, palomitas o refrescos. 
Ventana 1:
La primera ventana sirve para ingresar tu nombre y contraseña y dejarte ingresar a las siguientes ventanas, si esta mal la contraseña o usuario no dejará continuar a las siguientes ventanas.
Ventana 2:
calcular la venta de boletos, elegir la película y el horario, elegir cuántos niños y cuántos adultos quieren boletos.
Ventana 3:
Esta Ventana es la de dulcería, esta calcula  el precio de cada cosa que tu ordenes, en esta puedes poner cuantas cosas quieres de cada producto y muestra que productos y combos ofrece.
Ventana 4:
En esta ventana aparecerá la suma total de todo lo que has pedido en las otra ventanas, como: cuántos boletos de niños y cuantos de adultos, cuantos refrescos, cuantas palomitas o cuantos combos.
